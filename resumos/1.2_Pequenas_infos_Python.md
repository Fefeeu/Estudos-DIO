# **Pequenas Adições**

## pass
"Ignora" uma execução
```Python
derf funcao_generica(variavel):
    pass    # essa palavra reservada 'pass' faz com que o Python "feche" a funcao
```

# match
Basicamente o switch case de outras funções
```Python
a, b = float(input().split())
operacao = str(input("+, -, *, /: "))
# EQUIVALENTE A SWITCH CASE
match operacao: 
    case "+":
        print(a + b)
    case "-":
        print(a - b)
    case "*":
        print(a * b)
    case "/":
        print(a / b)
```
